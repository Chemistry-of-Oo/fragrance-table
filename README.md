# Oo Olfactive Taxonomy

An open classification system for fragrance accords, developed by Oo La Lab.

## What it is
The Oo Olfactive Taxonomy defines 27 discrete olfactive accords — organised by 
molecular weight from lightest to heaviest — as a shared vocabulary for describing, 
comparing, and encoding fragrance compositions.
